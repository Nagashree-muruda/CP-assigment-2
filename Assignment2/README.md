# HTML Authentication without CSS and JS
This project demonstarates a simpe authentication using only HTML

## Pages Included
1. home.html
2. login.html
3. register.html
4. forgot-password.html
5. reset-password.html
6. dashboard.html

## How pages navigation works

1. **Home Page**
   - Navigate to login or register page

2.  **Register Page**
     - After registration Redirects to Login

3.  **Login Page**
   - After login Redirects to Dashboard
   - Forgot Password Redirects to Forgot Password page


4. **Forgot Password Page**
   - Redirects to Reset Password page

5. **Reset Password Page**
   - After reset password Redirects to Login page

6. **Dashboard Page**
   - Shows SUccesfully Logged in.
   -  Has Logout  button to Logout and to Redirect Login page

## Technologies Used
    - HTML5 only
    - Anchor tag based redirection
    - No JavaScript  
    - Bootstrap 5 (CDN)
    - CSS3 (Custom Styling)
    - Bootstrap Icons

## UI Enhancements
1. Gradient animated background
2. Glassmorphism card design
3. Styled buttons with hover effects
4. Improved typography using Google Fonts (Poppins)
5. Responsive layout using Bootstrap classes

## The application is fully responsive and works smoothly on:
Desktop 
Laptop 
Tablet 
Mobile 


## How to Run
1.  clone the repository.
2. Open `home.html` in any web browser.
3. Navigate through the system using the provided links.
